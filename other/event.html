event对象的属性如下：

target属性:(显示哪个 DOM 元素触发了事件)
event.target.nodeName：触发事件的节点名称

js代码：

  $("button").click(function(event){
    $("div").html("点击事件由一个 " + event.target.nodeName + " 元素触发");
  });

html代码：

结果输出：

    点击事件由一个 BUTTON 元素触发
ps:IE下,event对象有srcElement属性,但是没有target属性;Firefox下,event对象有target属性,但是没有srcElement属性.但他们的作用是相当的;
即：firefox 下的 event.target = IE 下的 event.srcElement
getElementById属性：（id）
var x=document.getElementById("myHeader")
getElementsByName属性：（name属性）
var x=document.getElementsByName("myInput");
getElementsByTagName属性：(标签)
var x=document.getElementsByTagName("input");
